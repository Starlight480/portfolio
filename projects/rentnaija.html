Full-Stack Web App

RentNaija

Hyper-local rental platform for Ajah, Lagos. Verified direct landlord listings with zero agent fees — cutting out the middleman.

Supabase Paystack Firebase Vercel Node.js
Role
Full-Stack Developer
Status
In Development
Stack
HTML / Supabase / Firebase / Paystack
Location
Ajah, Lagos, Nigeria

The Problem

Renting in Lagos is broken. Tenants pay 10-15% of annual rent as agent fees — that's ₦150K+ on a ₦1M apartment — for zero guarantee. Listings are unverified, landlords are unreachable, and fraud is rampant. The Ajah corridor, a fast-growing student-heavy suburb, is especially underserved.

The Solution

RentNaija eliminates the agent entirely. Every landlord is physically verified. Every listing is confirmed. Tenants browse, contact landlords directly, and pay zero agent fees. The platform handles discovery, verification, and payments — not intermediation.

Verified Landlords

Physical verification process ensures every landlord is real and every listing is legitimate.

Zero Agent Fees

Direct landlord-tenant connection. No middleman, no hidden costs, no pressure tactics.

Paystack Integration

Secure rent payments and deposit escrow via Paystack. Full transaction history for both parties.

Real-Time Notifications

Firebase Cloud Messaging powers instant alerts for new listings, messages, and verification updates.

Tech Architecture

Layer Technology Purpose
Frontend HTML / CSS / Vanilla JS Static site, fast loads, mobile-first
Hosting Vercel Edge deployment, zero config
Auth Supabase Auth Email + phone OTP, session management
Database Supabase PostgreSQL Landlords, properties, listings, users
Notifications Firebase Cloud Messaging Push notifications for listings & messages
Payments Paystack Rent payments, deposit escrow

Design Approach

Dark glassmorphism UI — deep backgrounds, frosted cards, emerald accents for trust, amber for CTAs. Every design decision serves the Nigerian market: high contrast for outdoor mobile use, large touch targets, minimal data usage. WCAG AA contrast ratios verified across all text.

Key Features

  • Landlord registration with ID verification
  • Property listing CRUD with image uploads
  • Search and filter by location, price, bedrooms
  • Direct in-app messaging (tenant ↔ landlord)
  • Paystack-powered rent payment flow
  • Admin dashboard for verification management
  • Featured listings for premium placement
  • View count and analytics per listing

Target Milestone

10 verified landlords listed before semester resumes (late July 2026). This is the critical mass needed to make the platform useful for incoming students and young professionals in the Ajah corridor.
